  • ¿Qué es BT21?

    • BT21 es la primera presentación de LINE Friends Creators, un proyecto formado para crear nuevos personajes para la serie Line Friends. La banda surcoreana BTS es el primer artista de este proyecto. El objetivo de esta colección es mostrar la conexión entre BTS y Line Friends en términos de popularidad en el mundo.
